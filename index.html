The Krojanty game

Krojanty is a strategy game developed by an eight-member IT student team as part of the ENSISA immersion program, completed over an intensive 15-day period. It features a custom C game engine, a GTK4 interface, an AI using minimax with alpha-beta pruning, and full client–server multiplayer. My contributions focused on implementing core game logic and designing the user interface, ensuring smooth gameplay interactions and a clean, functional layout. The project uses a structured architecture, Makefile-based compilation, unit testing, and Doxygen documentation to maintain professional-quality development.

Weather app

This Weather App allows users to check the current weather and a 5-day forecast in 3-hour intervals for any city. It was developed entirely using JavaScript and integrates the OpenWeatherMap API to fetch real-time data such as temperature, humidity, wind, and weather conditions. The interface updates dynamically based on user input, providing clear and responsive information. The project demonstrates practical use of APIs, asynchronous requests, and DOM manipulation to create an interactive web application.
