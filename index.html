Transport network

This Transport Network project models a city’s public transport system (bus, metro, tram) as a graph and provides interactive features for analysis and route finding. Developed entirely in C, it implements adjacency lists, dynamic arrays, and hash tables, with Dijkstra’s algorithm for shortest paths and multiple sorting algorithms to analyze station connectivity. The program features a command-line menu for station info, neighbor listing, minimal path calculation, and degree-based sorting, following a modular architecture, Makefile-based compilation, and clean coding standards for maintainability and clarity.
